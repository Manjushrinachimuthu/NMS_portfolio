# Manjushri N — Personal Portfolio

A modern, fully responsive developer portfolio built with **Next.js 14**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**. Designed to showcase projects, skills, certifications, and experience with smooth animations and a clean dark aesthetic.

---

## ✨ Features

- **Animated Hero Section** — Typewriter role animation, floating badges (Open to Work, Location), orbital ring visual, and live stats
- **About Section** — Personal intro, internship experience cards, and skill chips
- **Education Timeline** — Academic background from SSLC to B.Tech with a vertical timeline layout
- **Skills Marquee** — Infinite scrolling dual-row marquee of 24+ technologies and tools
- **Certifications** — Interactive selector with animated detail cards for 6 certifications
- **Featured Projects** — 3 projects with custom animated visuals (Smart Price Checker, AI Pathfinder, NoteX AI)
- **Contact Section** — Working contact form with email, LinkedIn, and GitHub links
- **Sparkle Background** — Subtle animated particle background across the entire page
- **Responsive Navbar** — Sticky nav with mobile hamburger menu and resume download link

## 🏃 Getting Started

```bash
# Clone the repository
git clone https://github.com/Manjushrinachimuthu/NMS_portfolio.git

# Navigate into the project
cd NMS_portfolio

# Install dependencies
npm install

# Run the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
